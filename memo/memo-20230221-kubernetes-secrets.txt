


### 시크릿 생성 명령어 생성 방법
kubectl create secret <TYPE> <NAME> [OPTIONS]

(1) 일반 시크릿 생성
kubectl create secret generic <NAME> \
[--from-file=[key=]source] \
[--from-literal=key1=value1]

kubectl create secret TYPE --from-listeral=KEY=VALUE
kubectl create secret TYPE SECRET --from-file=FILE
kubectl create secret TYPE SECRET --from-file=KEY=FILE

(2) 도커 저장소 인증 정보용 시크릿 생성
kubectl create secret docker-registry <NAME> \
--docker-username=user --docker-password=password \
--docker-email=email [--docker-server=string]

(3) TLS 키/인증서용 시크릿 생성 기법

---
kubectl create secret generic secret-generic1 \
--from-literal=secretkey1=secretvalue1
